# ミッドポイント計算アプリ

占星術における「ミッドポイント（2天体の中間点）」を簡単に計算できるスマートフォン向けWebアプリです。

---

## 🔗 公開URL

https://akitai1117.github.io/midpoint-app/midpoint-app/index.html

---

## 📱 概要

本アプリは、占星術で使用される天体の位置（星座・度・分）を入力することで、  
2つの天体のミッドポイントを自動で計算・表示するツールです。

スマートフォンからの利用を前提に設計しており、  
直感的に操作できるシンプルなUIを目指しました。

---

## ✨ 主な機能

- 9天体（太陽〜冥王星）から2つを選択
- 星座（12サイン）・度・分を入力
- ミッドポイントを自動計算
- 結果を「星座 + 度 + 分」で表示
- 入力エラーの検出・表示
- スマートフォン対応デザイン

---

## 🧮 計算ロジック

- 星座を0〜360度の黄経に変換して計算
- 円周（360度）を考慮し、近い側の中点を採用
- 差が180度の場合は計算不可としてエラー表示
